## 🚀 How to Deploy (Choose One)

### Method 1: GitHub + Vercel Dashboard (Recommended)

**Time: 5-10 minutes**

```bash
# Step 1: Push to GitHub
git init
git add .
git commit -m "Initial commit - Techwise Portfolio"
git remote add origin https://github.com/YOUR_USERNAME/techwise-portfolio.git
git push -u origin main
```

**Step 2: Deploy on Vercel**
1. Go to [vercel.com](https://vercel.com)
2. Sign up/Login with GitHub
3. Click "Add New Project"
4. Import your repository
5. Click "Deploy"
6. ✅ Done! Your site is live!

---

### Method 2: Vercel CLI (For Developers)

**Time: 2-3 minutes**

```bash
# Install Vercel CLI
npm install -g vercel

# Login
vercel login

# Deploy to production
vercel --prod
```

✅ Done! Your site is live!

## 🆘 Troubleshooting

### Build Fails
```bash
rm -rf .next node_modules
npm install
npm run build
```

### Images Not Loading
- Check `public/gg.jpg` exists
- Verify image paths in components
- Check `next.config.ts` image configuration

### Links Not Working
- Verify all URLs are correct
- Check for typos in href attributes
- Test in incognito mode
